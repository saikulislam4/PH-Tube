Discuss the scope of var, let, and const

var: var is a functional scope,var can be re-declared and re-assign, if we use var then the project will cause problems

let: let is a global scope variable, let can be re-assign but let can't re-declared

const: const is a also global scope variable, but const can't' re-assign and can't' re-declared
